# CAD Feature Selection

This repository contains the implementation of the feature selection framework proposed in:

Foad Ghasemi et al.
"Feature Selection in Pre-Diagnosis Heart Coronary Artery Disease Detection: A Heuristic Approach Based on Information Gain Ratio and Gini Index"
ICWR 2020.

## Dataset
Z-Alizadeh Sani Dataset

## Methods
- Information Gain Ratio
- Gini Index
- Feature Ranking
- Feature Selection

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
